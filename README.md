# Hollywood Movies Dashboard (2006-2016)

This PowerBI dashboard provides insights into the 1000 best Hollywood movies released between 2006 and 2016. The dashboard allows users to explore various aspects of these movies, including box office performance, genre and ratings.

## Overview

This project includes a PowerBI dashboard that visualizes data for the top 1000 Hollywood movies released over a decade (2006-2016). The dataset includes information such as movie titles, genres, release year, box office earnings, ratings, and more. The dashboard enables users to analyze trends, compare movie performances, and gain insights into the film industry during this period.

## Features

- **Top Movies by Box Office:** Visualizes the highest-grossing movies of each year.
- **Genre Analysis:** Breaks down movies by genre to see which types of films were most popular.
- **Yearly Trends:** Shows trends in movie production, earnings, and ratings over the years.
- **Filter Options:** Allows users to filter data by year, genre, ratings, and other parameters.

## Data Source

The data used in this dashboard is sourced from a Kaggle dataset containing IMDB data. It includes the following fields:

- **Title:** The name of the movie.
- **Release Year:** The year the movie was released.
- **Genre:** The primary genre(s) of the movie.
- **Box Office Earnings:** The total revenue generated by the movie.

## Business Use Cases

This dashboard can be valuable in various business scenarios, including but not limited to:

### 1. **Market Research for Film Production:**
   - **Trend Analysis:** Producers and studios can use the dashboard to identify genre trends, audience preferences, and box office success factors over the years. This can guide decisions on the types of films to greenlight.
   - **Competitive Analysis:** Understand how competitors' movies have performed in the past and identify key factors that contributed to their success.

### 2. **Content Acquisition for Streaming Platforms:**
   - **Library Expansion:** Streaming services can use the dashboard to identify which types of movies to acquire based on historical popularity and audience ratings.
   - **Content Curation:** Curate movie collections that align with current viewer preferences, increasing user engagement on the platform.
